## Deployment

This project can be deployed using:

- GitHub Pages  
- Netlify  
- Vercel  
- Any static hosting provider  

To deploy via GitHub Pages:

1. Push repository to GitHub  
2. Go to Settings → Pages  
3. Select branch  
4. Publish
